## Estrutura

- `index.html` na raiz como entrada principal do site
- `pages/` com as páginas institucionais e do blog
- `assets/css/` para estilos globais
- `assets/js/` para scripts
- `assets/img/` para todas as imagens do site
- `admin/` para o painel administrativo
- `financeiro/` para páginas específicas da área financeira
